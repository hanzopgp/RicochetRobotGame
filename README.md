## How to use :

- Execute : ``java -jar dist/RicochetRobot.jar``

## How to play :

- select a robot : "R", "G", "B", "J"
- make a move : up/down/left/right arrows
- resolve : "S"
- new game : "Enter"

## Details :

- Implementation of MVC
- Implementation of A*
- Implementation of DFS
